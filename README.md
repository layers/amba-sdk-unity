# Amba Unity SDK

The Amba Unity SDK (`com.layers.amba`) for Unity Package Manager.

## Install

In Unity, open **Window → Package Manager → + → Add package from git URL**:

```
https://github.com/layers/amba-sdk-unity.git
```

## Docs

- Quickstart: <https://docs.amba.dev/docs/sdk/unity>
- API reference: <https://docs.amba.dev/docs/sdk/features>
- Issues + source: this repo is a consumption mirror; development happens in the private dev monorepo and tagged releases are pushed here.
